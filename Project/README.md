This repository contains my ECEN 603 – Logic Design Using HDL course project at Santa Clara University, where I designed and verified a packet router in Verilog. The project implements a modular routing system with packet registration, destination decoding, FIFO-based buffering, output control logic, and a finite state machine to manage packet loading, forwarding, and parity/error handling. The design was simulated and synthesized to evaluate functional correctness, timing, and area, and the repository includes the HDL source files, testbenches, synthesis scripts, and supporting project documentation.
