# ECEN603-Logic_Design_Using_HDL
This repository contains my ECEN 603 – Logic Design Using HDL projects, implemented in Verilog/SystemVerilog at Santa Clara University. The goal of these projects is to practice digital system design using hardware description languages, including combinational and sequential logic, finite state machines, datapath design, modular HDL development, simulation, and verification with testbenches. The repository includes source files for the project implementations as well as any helper modules, testbenches, scripts, and supporting documentation. This project is shared for personal reference and portfolio purposes only. If you are currently enrolled in ECEN 603, please do not copy this work or submit it as your own. Follow SCU’s academic integrity policies.
